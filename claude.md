# Claude Project Guidelines

## UI/UX & Design Consistency
- Maintain the current design and theme engine when adding new UI components or editing existing ones.
- Ensure consistency across all user flows and interactions.

## Frontend Development
- Use the `frontend-design` skill for all UI development tasks to ensure cohesive, high-quality interfaces.
- Treat all UI elements as part of a unified system; avoid unsupported customizations that may break the theme engine.

## Senior Fullstack Engineering Practice
- Approach all tasks as a senior fullstack engineer, prioritizing security, performance, and user experience.
- Follow best practices for code structure, security measures (e.g., input validation, secure API calls), and UX design.

## Development Workflow
- Test all features thoroughly before advancing to new tasks. Verification steps include:
  - Functional testing
  - Security checks (e.g., vulnerability scans)
  - Cross-browser compatibility
  - Performance benchmarks
- Document and verify each feature's implementation in relevant code files.

## Additional Instructions
- Always reference the current theme engine documentation when making design decisions.
- Collaborate with the frontend team for complex UI patterns.
- Review and update this guide as new design/UX requirements emerge.
